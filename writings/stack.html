Built in 2 days as a weekend project at Dev Bootcamp, StackToldYouSo is a spoof off of the ever relied-upon StackOverflow. The project's objectives were to build a Ruby on Rails app that uses polymorphic relationships and to write tests. Lots of tests.

Technologies used: Ruby on Rails, HerokuApp, HTML5, CSS3, Bootstrap, rSpec, CapyBara, CircleCi, FactoryGirl

App Features:

  • User can post a question.
  • User can comment or answer a question.
  • User can vote on quality of questions/answers.
  • User can only comment on answers if they are logged in.
  • Answers are displayed based on number of votes.
